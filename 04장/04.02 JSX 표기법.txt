
SPA에서 HTML 파일은 한 개뿐이다.

리액트는 JSX 표기법을 사용한다.
JSX 표기법에서는 함수의 반환값으로 HTML 태그를 기술할 수 있고 그것을 컴포넌트로 다뤄 화면을 구성한다.
const App = () => {
  return <h1>안녕하세요.</h1>;
};

JSX에서 return 이후는 한 개의 태그로 둘러싸여 있어야 한다. 빈 태그로 감쌀 수도 있다.
const App = () => {
  return (
    <>
      <h1>안녕하세요.</h1>
      <h2>반갑습니다.</h2>
    </>
  );
};
이 표기법은 불필요한 DOM을 생성하지 않는다.

리액트에서는 함수명을 HTML과 같이 태그로 감싸서 컴포넌트로 다룰 수 있다.
함수로 정의된 컴포넌트를 함수 컴포넌트라 부른다.
과거에는 Class로 정의된 클래스 컴포넌트를 사용했지만, 최근에는 함수 컴포넌트를 주로 사용한다.
